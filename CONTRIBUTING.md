# Contributing to Imposter Game Words

First off, thank you for considering contributing to Imposter Game Words! It's people like you that make our word database the best on the internet.

## 📝 Suggesting New Word Pairs

The core of [ImposterGameWords.com](https://impostergamewords.com) is our carefully curated word pairs. 

### What makes a good word pair?
A good word pair must have **overlapping attributes** but **distinct core identities**.
- ❌ **Bad (Too Similar):** Couch / Sofa (Basically the same thing, impossible for civilians to prove they know the difference without giving it away).
- ❌ **Bad (Too Different):** Ocean / Laptop (Too easy for civilians to expose the imposter instantly).
- ✅ **Good:** Airport / Train Station (Both are transit hubs, have tickets, waiting areas, and luggage, but have distinctly different vehicles and security protocols).

### How to submit:
1. Navigate to the **Issues** tab.
2. Click **New Issue**.
3. Select the **New Word Suggestion** template.
4. Provide the word pair and suggest which category it belongs to (Standard, Kids, Tech, etc.).

## 🐛 Reporting Bugs

If you find a bug on the main website (UI glitch, broken link, generator logic error):
1. Navigate to the **Issues** tab.
2. Select the **Bug Report** template.
3. Include your device type, browser, and steps to reproduce the issue.

## 🧑‍💻 Contributing to the Tools / Scripts

If you want to improve our Python utility scripts (`check_links.py` or `build_pages.py`):
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/improve-link-checker`).
3. Make your changes.
4. Submit a Pull Request.

We review PRs weekly. Thank you for your support!
